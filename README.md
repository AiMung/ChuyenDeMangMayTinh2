## 📌 Giới thiệu Repository
Đây là nơi lưu trữ các bài tập thực hành (Labs) và đồ án môn học **Chuyên đề Mạng máy tính 2**. Nội dung tập trung vào việc nghiên cứu các hệ thống mạng P2P (Chord, CAN, BitTorrent), quản lý mã nguồn với Git nâng cao và triển khai hạ tầng trên môi trường Linux.

## 📁 Cấu trúc dự án
- `Lab01-GitKatas/`: Tổng hợp 16 bài tập thực hành Git nâng cao (Basic Commits, Rebase, Merge, Stashing...).
- `Reports/`: Chứa file báo cáo chi tiết (.docx/.pdf) và hình ảnh minh chứng thực hiện trên môi trường Ubuntu 22.04 Minimal.
- `Theory/`: Các bài tìm hiểu về cấu trúc mạng P2P có cấu trúc và không cấu trúc.

## 🛠 Môi trường thực hiện
- **OS:** Ubuntu 22.04 LTS (Minimal Installation)
- **Virtualization:** VMware Workstation
- **Tools:** Git CLI, Nano Editor, Markdown, Personal Access Token (PAT) for Authentication.
